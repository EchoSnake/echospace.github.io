Ubuntu老电脑焕发新生活

2025/11/20 Linux Linux Ubuntu 博客 AI 服务器 操作系统 Docker 废物利用 老电脑

引言

这是一台戴尔灵越5488，2018年9月发布，搭载4核8线程的Intel i7-8565U处理器，配备8GB内存。在2025年的今天，虽然硬件规格不算顶尖，但通过安装Ubuntu系统，完全可以焕发它的第二春，重新获得流畅体验，适合办公、学习甚至轻度服务器使用。

开始安装Ubuntu

  1. 制作启动U盘
    将下载的Ubuntu 24.04镜像使用Balena Etcher烧录到U盘，作为启动盘。

  2. 进入BIOS设置
    重启电脑，进入BIOS或启动菜单（一般按F11或F12），选择U盘启动。

  3. 安装系统
    进入Ubuntu安装界面后，按提示操作：

  • 选择语言和键盘布局。

  • 建议此处先不联网。

  • 磁盘选择：如果没有重要文件，建议格式化整个磁盘安装，若需保留数据务必提前备份。

  • 设置用户名和密码。

  • 确认安装开始。

  1. 安装完成并重启
    安装完毕后，立即拔出U盘，系统自动重启进入已安装的Ubuntu。

系统初步配置

  1. 联网更新系统

打开终端，执行这条命令进行组件更新和清理冗余：

sudo apt update sudo apt upgrade -y sudo apt autoremove -y
  1. 安装星火商店

推荐使用星火商店简化软件管理，方便安装各种应用。下载安装包后，使用这条命令进行安装：

sudo apt-get -y install spark-store_4.8.2_amd64.deb

安装完成后，即可在软件中搜索软件进行一键安装使用。

  1. 安装常用软件

办公套件推荐OnlyOffice。若在星火商店没有找到你需要的对应客户端，大部分的软件都有 Linux 的版本。
官网都可以下载到对应 .deb 安装包，结合系统自带的 apt-get 一键安装即可。

系统优势与扩展应用

Ubuntu相比Windows更轻量，能显著提升老旧机器的运行流畅度。你还可以将电脑变为家庭服务器：

  • 配置Samba文件服务器：轻松共享文件和资料。

  • 部署Docker容器：运行多个轻量级服务，提高硬件利用率。

  • 远程办公与开发环境搭建：用Linux工具链带来更高效的生产力。
