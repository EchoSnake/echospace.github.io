扩展屏幕–手机平板做副屏是怎么实现的

扩展屏幕–手机平板做副屏是怎么实现的，跟着我操作

准备工具

在我的博客找到这两个软件下载：

  • **ParsecVDisplay**：用于生成虚拟屏幕
  • UU远程：可以用任意远程软件替代
    • 电脑版
    • 安卓版
    • IOS版本
    • MACOS版本

装虚拟显示器

  1. 打开安装包
  2. 点击同意继续
  3. 等它装完
  4. 安装好后会自动打开，点击中间的添加屏幕

装UU远程

主机端（电脑）：

  1. 打开UU远程安装包，安装
  2. 打开后输入手机号收验证码登录

副设备（手机/平板/笔记本）：

  1. 下载UU远程，登录同一个账号
  2. 找到电脑，点击连接
  3. 点击右上角图标，启用按键映射（显示鼠标指针）
  4. 显示里选择到显示屏2

开始用

  1. 连好后回到桌面
  2. 在桌面右键显示设置，将屏幕2拖到你实际拜访的位置方便使用
  3. 完成

总结

有问题评论区见，觉得有用一键三连！
