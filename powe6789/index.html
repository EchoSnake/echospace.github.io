Windows系统增强神器PowerToys

2025/11/9 Windows Windows效率神器 PowerToys 微软官方工具 免费开源 电脑技巧 程序员必备 设计师必备 前端必备

三星有 GoodLock，Windows 也有自己的外挂——PowerToys！
免费、开源、零广告，装完直接发现新大陆！
GitHub 搜 PowerToys，或者在微软商店中也可以一键下载,网络不支持的同学,也可以在我的博客自取.

点击此处下载

这软件是微软亲儿子，稳得一批。 花几分钟时间,带你看完 9 个我个人觉得最神级功能， 程序员、设计师、办公党全部原地升天！

① 高级粘贴 | Win+Shift+V（0:23-0:43）

直接复制代码,会有方方面面的格式问题,
使用粘贴 json markdown 就可以一键自动排版,
所有复制的图片都可以直接用.png格式粘贴,
写代码，爽到飞起！
他还自带转码:
MP4 想抽音频？顺手选 MP3 粘贴，批量转码省掉格式工厂。

② 颜色选择器 | Alt+C（0:43-0:58）

设计师看过来！
任何时候见到好看的颜色,直接呼出 颜色选择器
左键单击你想要的颜色格式应有尽有
比PS吸管,微信QQ截图都快不少.

③ PowerToys Run | Alt+Space（0:58-1:10）

Win11 搜索卡顿还时不时资源管理器未响应？
Alt+Space 毫秒呼出，拼音首字母就能开软件，
路径直达，省去大量找图标的时间。

④ 屏幕标尺 | Ctrl+Shift+R（1:10-1:23）

前端牛马神器！
横竖随便拉，px像素、cm厘米、inch英寸 一键切换，
数值左键自动复制，不用再去PS里使用标尺慢慢比对。

⑤ 文本提取器 | Win+Shift+T（1:23-1:35）

系统级 OCR，框一下，图片、PDF、视频字幕一秒出文字，
免费不限量，第三方 OCR 软件可以卸载了。

⑥ 窗口裁剪器（1:35-1:45）

查教程必备！
框选任意区域，一键生成置顶悬浮窗，
左边挂代码，右边写笔记，Alt+Tab 切的自己都不知道哪个窗口是干嘛的？再也不存在了.

⑦ FancyZones（1:45-2:00）

分屏狂魔！
按住 Shift 拖窗口，4K 屏切成三联、四联、六联，
模板还能自定义，比 Win11 自带分屏高到不知道哪里去了。

⑧ 查找我的鼠标（2:00-2:10）

星际党福音！
双击左 Ctrl，屏幕瞬间缩小高亮鼠标，
还能设白名单，打游戏时自动屏蔽误触。

⑨ 资源管理器预览（2:10-2:20）

开启后，.md、.svg、.pdf 等格式都可以直接预览，
搭配「速览」使用 ctrl + space 快速查看，文件管理直接起飞。
