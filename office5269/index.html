Microsoft Office安装激活教程

怎么装Office，怎么激活？很简单。

下载工具

  • 进入这个网页
  • 点击下载，如果下载不下来就换后面的地址
  • 左键单机下载下来的压缩文件，点击上方全部解压缩，点击提取
  • 在弹出窗口的文件夹中找到 office tool plus.exe
  • 双击打开它，若有提示选择 同意

安装

  • 选择左边的部署
  • 系统里自带的Office如果没有激活，点击这个垃圾桶进行卸载它
  • 点击添加产品
  • 推荐选择Office 2021 专业版
  • 选择你想要的组件，我这里勾选的是幻灯片，表格，文本文档，其他组件按照你自己的需求
  • 点击开始部署
  • 选择 是
  • 期间会弹出窗口，什么都不用管，等他自己安装完成

  • 显示 一切已就绪 字样，点击关闭

激活

  • 右键开始菜单，选择”Windows PowerShell(管理员)”
  • 输入命令后按回车
    irm https://get.activated.win | iex
  • 在弹出的窗口按2
  • 这个窗口再按1
  • 激活完成后会这样提示，点叉叉关闭这些窗口

检查激活

  • 找到word并打开
  • 点接受
  • 点击左下角账户，就可以查看到已经激活

总结

安装+激活，两步搞定Office！永久激活，一次到位。有问题评论区见~
