Windows系统安装完整指南

2025/11/23 Windows 博客 Windows 装机 装系统 自维修 Windows10 Windows11

所有的内容我都将放在我的博客中,可以在博客中对照着进行操作,所有需要的软件以及工具都可以在我的博客中进行下载.

[微软U盘制作工具](Media Creation Tool)

各个品牌官方驱动网址
微星 华硕 技嘉 华擎 联想 戴尔 惠普

01｜准备工具：只要一个U盘

要求：

  • 8GB 以上 U 盘（用于制作系统的引导盘,会进行格式化,有数据做好备份工作）

  • 一台能正常上网的电脑(用于下载系统镜像和制作引导U盘)

  • 想安装系统的那台机器（废话但必须说，因为有人真的会问🤣）

  • 提前下载好

    • 微软U盘制作工具
    • 对应的驱动,特别是网卡

02｜制作系统U盘：微软官方工具最稳

小白不要乱下“XX装机大师”,”XX一键装机”等一系列软件。

直接用微软官方的：

下载入口

打开浏览器,搜索 Windows11

进入微软官网 → 搜索 “Media Creation Tool”。

步骤非常顺滑：

  1. 打开工具

  2. 选你要安装的 Windows 版本

  3. 按提示选择 “U 盘”

  4. 等待进度条拉满

结束后，你的 U 盘就变成了“系统安装盘”。

在我的博客中点击对应链接,进入官网,使用对应型号下载网卡驱动,组装机就进入主板品牌中下载网络驱动

03｜进入 BIOS 设置 U 盘启动

找到你电脑上的

常见按键：

品牌 进入 BIOS 启动菜单（Boot Menu）
联想 Lenovo（ThinkPad） F1 F12
联想 Lenovo（IdeaPad） F2 F12
华硕 ASUS F2 ESC
惠普 HP F10 ESC 或 F9
戴尔 Dell F2 F12
宏碁 Acer F2 F12
微星 MSI（笔记本） DEL F11
机械革命 MECHREVO DEL F7
雷神 ThundeRobot F2 F12 或 F7
神舟 Hasee F2 F12
小米 / RedmiBook F2 F12
华为 MateBook F2 F12
荣耀 MagicBook F2 F12
三星 Samsung F2 F10 或 F12
Surface（部分机型） 长按音量+ 开机 不支持传统 Boot Menu
直接在 Boot 选项中选择对应的 USB 启动项
或者：
进入 BIOS 后找到引导选项：
Boot → 把 USB 放第一位
保存退出（Save & Exit）。

电脑这时会从 U 盘启动。

04｜开始安装系统（正式环节）

启动后会看到蓝色的 Windows 安装界面。

你确保自己只干两件事：

① 选择 “自定义安装（高级）”

让你能看到磁盘分区。

② 选择要安装系统的盘

一般是：

  • C盘

  • 或者容量 100GB 左右的系统盘

如果小白懒得分区，直接格式化原系统盘就好，别乱动数据盘。

然后点 下一步 —— 然后就是等。

（期间喝杯水、刷个视频，不要怀疑人生，进度条动得很慢很正常。）

05｜进入桌面：装驱动，解锁满血性能

装完系统只是“点亮屏幕”，真正的满血是：

1.安装显卡驱动

Nvidia:

进入 Nvidia 驱动下载页面 —— 选择到自己对应的显卡驱动进行下载，然后安装即可。

AMD:

进入 AMD 驱动下载页面，选择自己对应的显卡驱动下载，安装即可。

INTEL:

进入 INTEL 驱动下载，输入自己对应的显卡型号进行下载，安装即可。

Tips:

笔记本的驱动建议都在对应品牌的官网进行下载，以免遇见兼容性或者部分快捷键组件失灵。

2.主板驱动

最简单方式：
各品牌官网 → 输入你电脑主板型号 → 下载所有的驱动进行安装。

3.组装机

如果你是组装机,那就按照你主板的品牌,显卡的品牌,进入官网,找到对应的驱动进行下载安装。

06 | 完成安装

右击 Win菜单键 点击 设备管理器，当没有感叹号的时候，就代表你的驱动已经完全安装完成可以开始体验了。
如果你有部分第三方的硬件或者是类似于加装的摄像头之类的外部配件，请前往对应硬件的品牌官网下载驱动。
